Ubuntu22, c++17, QTDBUS를 활용하여 BLE5.x 통신 시스템 구축.
프로젝트 명 : robot_ble_communicator
각 로봇은 255 Byte내 Advertising 데이타를 주기적으로 만들어 Advertise 한다.
Advertising 데이타 : robot_id, 현재시각(UTC Timestamp), float (x, y, z, yaw, current_speed), 
  task start time(UTC), guide graph id, goal vertex_id, path_length, path list of pair{vertex_id, arrival time} upto 40,
  CRC(2byte)
robot_id : 10 charaters
guide graph id : long
vertex_id : short
arrival time : unsigned short. max 650초. 0.01 sec = 1

Advertiser, Listener 구축
Advertising 주기 : 2회/초
실제 광고 오브젝트 등록(DBus Custom Advertising Object), 역직렬화/CRC 검증, 경로/상황 데이터 파싱 등 확장 로직 추가
h file 과 cpp 파일로 짝지어 구성
